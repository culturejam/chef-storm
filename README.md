# storm chef cookbook

Storm: a distributed realtime computation system

## Overview

This cookbook configures a machine to be a storm cluster.

## Recipes 

* `topology`    - Download and start a topology


## Integration

Supports platforms: ubuntu

Cookbook dependencies:

* java
* leiningen
* maven
* silverware
* install_from
